# 生成游戏美术资源

## 执行步骤

1. **构建 Prompt：** 根据用户需求，用**中文**构建一个高度描述性的 prompt，包含艺术风格、光照、氛围、构图等关键词。

   **Prompt 结构：** `[主体描述], [美术风格], [光照/渲染], [氛围/色调], [构图/视角]`

   **示例 Prompt：**
   `星际穿越，黑洞，黑洞里冲出一辆快支离破碎的复古列车，强视觉冲击力，电影大片，末日既视感，动感，对比色，oc渲染，光线追踪，动态模糊，景深，超现实主义，深蓝，画面通过细腻的丰富的色彩层次塑造主体与场景，质感真实，暗黑风背景的光影效果营造出氛围，整体兼具艺术幻想感，夸张的广角透视效果，耀光，反射，极致的光影，强引力，吞噬`

2. **执行生成脚本：** 使用 Bash 工具运行脚本。将 `[YOUR_PROMPT]` 替换为中文 prompt，可选提供输出文件名。

```bash
# 基础用法（自动生成文件名，如 art_17390000000.jpeg）
node .cursor/skills/generate-game-art/scripts/generate.js "[YOUR_PROMPT]"

# 指定输出文件名
node .cursor/skills/generate-game-art/scripts/generate.js "[YOUR_PROMPT]" "pixel_card.jpeg"
```

3. **处理结果：** 脚本会自动调用 API、等待图片生成，并下载到 `public/assets/` 目录。

4. **告知用户：** 显示下载的资源路径给用户。

## Prompt 撰写指南

- 用**中文**撰写，越详细越好
- 明确指定美术风格（像素风、写实、卡通、赛博朋克等）
- 描述光照和渲染效果（光线追踪、体积光、景深等）
- 指定色调和氛围（暗黑、明亮、冷色调等）
- 包含构图信息（广角、俯视、特写等）
- 避免使用"扑克牌"等词汇，改用"魔法卡牌"可绕过内容审核
